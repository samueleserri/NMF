# NMF

This repository contains the implementation, experiments and results of my bachelor thesis on Non-negative Matrix Factorization (NMF).


## Repository structure
- data/            — datasets used for experiments 
- notebooks/       — Jupyter notebooks for exploratory analysis and visualizations
- src/             — implementation of NMF algorithms and utilities
- experiments/     — experiment configurations and result logs
- results/         — processed outputs, plots and tables used in the thesis
- docs/            — thesis chapters, figures and supplementary material

## Requirements
- Python 3.8+
- Common libraries: numpy, scipy, scikit-learn, matplotlib, 
jupyter

# Install common dependencies
